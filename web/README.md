# Web UI for Layout QuickCheck

## Setup

### Install UI Requirements

From `web/ui`
```bash
npm install
```

### Install Backend Requirements

From repo root:
```bash
pip3 install -r requirements.txt
```

## Run

### Run the UI

From `web/ui`
```bash
npm start
```

### Run the Backend Server

From `web/server`
```bash
python main.py
```
